# Sisyphus

基于希腊神话的 2D 爬坡小游戏：操控西西弗斯推石上山，通过 QTE 蓄力与判定条完成攀登。

## 技术栈

- TypeScript
- Vite
- HTML5 Canvas

## 如何运行

### 环境要求

- Node.js（建议 18+）
- npm

### 安装依赖

```bash
npm install
```

### 开发模式（热更新）

```bash
npm run dev
```

浏览器打开终端里显示的本地地址（一般为 `http://localhost:5173`）即可游玩。

### 构建与预览

```bash
npm run build
npm run preview
```

`build` 会生成 `dist/`，`preview` 用于本地预览打包后的效果。
